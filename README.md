# Evotix Ticket Website

Evotix adalah platform web untuk pemesanan tiket acara secara online. Website ini dirancang untuk mengatasi masalah sistem konvensional seperti antrian panjang, risiko kehilangan tiket fisik, dan keterbatasan informasi. Pengguna dapat melihat detail acara dan memesan tiket dengan cepat, efisien, dan aman.

## Fitur Utama
- **Beranda**: Informasi singkat dan banner promosi event.
- **Detail Acara**: Tema, lokasi, waktu, dan pengisi acara.
- **Pemesanan Tiket**: Form online untuk memilih tiket dan melakukan pemesanan.
- **Dokumentasi Event**: Foto/video dari event sebelumnya.
- **Kontak Kami**: Form saran, pertanyaan, atau kerja sama.
- **Desain Responsif**: Nyaman diakses dari HP, tablet, maupun laptop.

## Tools & Teknologi
- Visual Studio Code
- HTML, CSS, JavaScript
- Bootstrap (Carousel, Tabs, Modal)
- GitHub Pages untuk hosting

## Implementasi
- Floating Search Bar untuk mencari event.
- Event Card Carousel menampilkan event populer.
- Sistem e-ticket dengan konfirmasi pemesanan.
- Tampilan ringan, mobile-friendly, dan fokus pada event hiburan lokal.

## Tujuan
- Mempermudah akses informasi event.
- Menyediakan sistem pemesanan tiket online yang praktis.
- Mendukung konsep **paperless** dengan e-ticket.
- Menarik minat pengunjung dengan desain modern dan responsif.
